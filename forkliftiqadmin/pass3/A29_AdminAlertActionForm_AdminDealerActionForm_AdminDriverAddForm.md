### Finding A29-7
**Severity:** MEDIUM
**File:** `AdminDriverAddForm.java`
**Location:** `validate` method, line 38
**Issue:** The `validate` method is a non-trivial public method that overrides `ActionForm.validate()` and contains business logic (empty-string checks on `first_name` and `last_name`; password equality check between `pass` and `cpass`). No Javadoc is present. A method of this complexity warrants documentation describing:
- What validation rules are applied.
- What `ActionErrors` keys are produced (`"first_name"`, `"last_name"`, `"pass"`).
- That message keys `"error.firstname"`, `"error.lastname"`, and `"error.pass"` map to the application's message resources.
- Potential NullPointerException risk: if `first_name`, `last_name`, `pass`, or `cpass` is `null` (all are initialised to `null` on lines 23–33), calling `.equalsIgnoreCase("")` will throw a `NullPointerException` at runtime. This logic defect should ideally be flagged in documentation (or a code-quality finding), but as a documentation issue the absence of any warning comment or Javadoc is MEDIUM severity.

---

### Finding A29-8
**Severity:** MEDIUM
**File:** `AdminDriverAddForm.java`
**Location:** `getDriverBean` method, line 61
**Issue:** The `getDriverBean(String sessCompId)` method is a non-trivial public factory method that constructs a `DriverBean` via a builder, injects the session-provided `companyId` (`sessCompId`), emits a debug log, and returns the bean. No Javadoc is present. Missing documentation includes:
- A description of the method's purpose (mapping form state to a domain bean).
- A `@param sessCompId` tag explaining that this parameter supplies the company ID from the HTTP session (not from any form field — the form contains no `companyId` field of its own).
- A `@return` tag describing the returned `DriverBean`.
- No mention that `op_code` is intentionally excluded from the builder call (lines 62–78 do not pass `op_code` to `DriverBean.builder()`), which may be intentional or an omission; without documentation it cannot be determined.
