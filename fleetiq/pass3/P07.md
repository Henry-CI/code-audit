### P07-F01: Missing file-level documentation for print_util_summary.jsp
**Severity:** MEDIUM
**Description:** The JSP file au_print/print_util_summary.jsp lacks file-level documentation explaining its purpose, which is to generate a printable vehicle utilization summary report. The file retrieves fleet vehicle data filtered by customer, site, department, and date range, and renders it in a tabular format. Without file-level documentation, developers cannot quickly understand the file's purpose, expected parameters, or its role in the reporting system.
**Fix:** Add a JSP comment block at the top of the file (after includes) documenting: (1) Purpose: "Generates a printable summary report of vehicle utilization by fleet number and model", (2) Required request parameters: gp_cd, user_cd, model_cd, st_dt, to_dt, loc_cd, dept_cd, form_cd, (3) Required session attributes: access_level, access_cust, access_site, access_dept, (4) Dependencies: Databean_report bean, UtilBean, (5) Output: Auto-prints HTML table and closes window on load.
**File:** au_print/print_util_summary.jsp
**Line(s):** 1
