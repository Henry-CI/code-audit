**Description:** The JavaScript function `open_report(url)` at line 8 in pages/admin/summary.jsp has no documentation comment. This function manages the opening of report windows and handles window state (checking if window is open, closing and reopening). Without documentation, developers may not understand the window management behavior or the expected format/content of the url parameter.
**Fix:** Add JSDoc comment before the function describing its purpose, the url parameter, and the window management behavior. Example:
```javascript
/**
 * Opens a report in a new popup window. If a report window is already open,
 * it will be closed and reopened with the new URL.
 * @param {string} url - The URL of the report to display
 */
```

**Method:** open_report
