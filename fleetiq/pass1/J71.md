### J71-F06: Session Fixation — Session Not Regenerated on Login
**Description:** In `chk_login()`, upon successful authentication the code calls `request.getSession(true)` (line 2042), which returns the existing session rather than invalidating it and creating a new one. ### J71-F11: chg_bms_pass() — SQL Injection via MD5 Calculation in SQL
**Description:** `chg_bms_pass()` executes MD5 computation inside the SQL query itself by concatenating the user-supplied plaintext password directly into the SQL string. This introduces SQL injection, and also means the plaintext password transits the database connection unencrypted.
