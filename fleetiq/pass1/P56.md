### P56-F03: Cross-Site Request Forgery (CSRF) - No Token Protection
**Description:** All forms in the reviewed files submit state-changing operations (update_user, delete_alert, unsubscribe) without any visible CSRF token protection. An attacker can craft a malicious page that submits these forms on behalf of an authenticated user, allowing unauthorized changes to user accounts, licenses, permissions, alerts, and subscriptions.

**Severity:** HIGH

**Checklist item:** 4. Session and CSRF
