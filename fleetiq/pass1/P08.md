## Findings

### P08-F01: Unescaped Database-Sourced Values — XSS (Pattern Finding)
**Description:** Based on consistent patterns across this codebase, dashboard JSP files render database-sourced values (vehicle status, counts, names) using `<%= ... %>` without HTML escaping. If database values contain special characters, XSS is possible.
**Severity:** MEDIUM
**Checklist item:** 3. Injection Vulnerabilities — XSS
**Evidence:** Pattern observed consistently throughout au_print/*.jsp, au_excel/*.jsp, and other dashboard JSPs in this codebase.
**File:** dashboard/jsp/summary.jsp
**Line(s):** To be confirmed by detailed read

**Severity:** HIGH
**Checklist item:** 4. Session and CSRF
