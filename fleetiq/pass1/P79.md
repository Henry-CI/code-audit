## Findings

### P79-F01: XSS vulnerability via JavaScript injection in onload attribute
**Description:** The gmtp_id request parameter is directly embedded into a JavaScript context in the body's onload attribute without proper encoding or escaping. An attacker can inject malicious JavaScript by providing a crafted gmtp_id parameter such as `');maliciousCode();//` which would execute arbitrary JavaScript in the victim's browser.
**Severity:** HIGH
**Checklist item:** 3. Injection Vulnerabilities - XSS (Cross-Site Scripting)
**Evidence:**
```jsp
Line 157: String gmtp_id = request.getParameter("gmtp_id")==null?"0":request.getParameter("gmtp_id");
Line 187: <body onload="set('<%=gmtp_id %>');">
```
**File:** reports/rpt_canvas_disp_ajax.jsp
**Line(s):** 157, 187

### P79-F07: No customer_id scoping visible in multi-tenant application
**Description:** This application runs on the multi-customer-sync-to-master branch, indicating multi-tenant functionality. None of the reviewed JSP files show explicit customer_id or tenant_id checks. While this scoping may occur in the DAO/bean layer, the lack of visible tenant isolation in files that handle driver data, vehicle data, and fleet operations is a concern. Without proper scoping, there is risk of IDOR vulnerabilities where one customer could access another customer's data.
**Severity:** HIGH
**Checklist item:** 2. Authentication and Authorization - Authorization and multi-tenancy
**Evidence:**
```
Files handling potentially multi-tenant data with no visible customer scoping:
- rpt_call_driver_licence_msg.jsp - handles driver license expiry (DriverDAO)
- rpt_check_driver_expiry.jsp - handles driver expiry alerts

- rpt_canvas_disp_ajax.jsp - displays vehicle canbus data by gmtp_id
- rpt_call_ftp_fleet_questions.jsp - uploads fleet questions via FTP

No visible checks like:
- session.getAttribute("customer_id")
- verification that gmtp_id/user_cd belongs to authenticated customer
```
**File:** rpt_call_driver_licence_msg.jsp, rpt_check_driver_expiry.jsp, rpt_check_driver_medical.jsp, rpt_canvas_disp_ajax.jsp, rpt_call_ftp_fleet_questions.jsp
