**Description:** Two files with backup/editor naming conventions are present in the `pages/admin/reports/` directory and are fully deployable by Tomcat: `unit_utilisation_report.jsp.old.jsp` (double `.jsp` extension — old backup of a production report page) and `unit_utilisation_report~.jsp` (tilde suffix — text editor backup). ### P43-F04: Reflected XSS via Unescaped `sub` Parameter in Navigation Links

**Description:** All ten files in this batch reflect the `sub` request parameter directly into `href` attributes within breadcrumb navigation links using `<%=sub%>` without any HTML encoding. A malicious actor can craft a URL with a `sub` parameter containing JavaScript (e.g., `javascript:alert(1)` or a closing-quote/HTML injection) that will be injected into the rendered page and executed in the victim's browser.
