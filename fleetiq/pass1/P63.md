### P63-F01: Critical SQL Injection via String Concatenation in Technician Data Fix Script
**Description:** The file 1_drivertech_fix.jsp contains numerous SQL injection vulnerabilities where user-controlled values from the database (technician IDs, customer codes, location codes, card IDs) are concatenated directly into SQL query strings without parameterization. An attacker who can manipulate technician user records or vehicle relationship data could inject arbitrary SQL to read, modify, or delete data across customer boundaries in this multi-tenant system.

**Severity:** CRITICAL

**Checklist item:** 3. Injection Vulnerabilities - SQL injection

**Evidence:**
```jsp
// Line 46-48: Technician list concatenated directly into query

query = "select driver_cd,vehicle_cd from fms_io_data where driver_cd in ("+techCDs+") group by driver_cd, vehicle_cd";

// Line 100: Vehicle code concatenated directly
query = "select \"USER_CD\",\"LOC_CD\" from \"FMS_USR_VEHICLE_REL\" where \"VEHICLE_CD\"="+entity.getVehCd();

// Line 109-111: Multiple values concatenated into complex query
query = " SELECT count(\"USER_CD\") from \"FMS_USR_MST\" where \"USER_CD\" = " + entity.getDriverCd() + " and (\"USER_CD\" in (select \"USER_CD\" from \"FMS_USER_DEPT_REL\" where \"CUST_CD\" = " + custCd + " and \"LOC_CD\" = " + locCd + ") OR \"USER_CD\" in (SELECT \"FMS_USR_MST\".\"USER_CD\" FROM \"FMS_USR_MST\", \"FMS_USER_DEPT_REL\"
                                                                                                                                                                                                                                      WHERE \"FMS_USR_MST\".\"USER_CD\" = \"FMS_USER_DEPT_REL\".\"USER_CD\" AND \"FMS_USR_MST\".\"ACTIVE\" = true
                                                                                                                                                                                                                                        and \"FMS_USR_MST\".\"USER_CD\" in ( select user_id from dealer_user_rel where dealer_id=(select dealer_id from \"dealer_cust_rel\" where cust_id = " + custCd + ")) and is_technician is true));";

// Line 118: String concatenation with user code
query = "select \"CARD_ID\" from \"FMS_USR_MST\" where \"USER_CD\" = '" + entity.getDriverCd() + "'";

// Line 126: ILIKE with concatenated card ID (potential wildcard injection)
```
