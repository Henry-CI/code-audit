### P75-F02: Untested Access Level Control Logic
**Severity:** CRITICAL
**Description:** Access level control logic is implemented in 8 of 10 JSP files (all except print_driver_licence_expiry_detail.jsp and print_hourmeter_exception.jsp initially, though the latter does have it). This security-critical functionality has zero test coverage. The logic includes session attribute retrieval, empty string checking, integer parsing, and bean configuration. No tests verify that unauthorized users are blocked or that access levels are correctly enforced.
**Fix:** Create dedicated test suite for access control:
1. Test with null session attributes
2. Test with empty access_level strings
3. Test with invalid access_level formats
4. Test each access level (0-5+) enforcement
5. Test with valid authorized users
6. Test with unauthorized users at each level
