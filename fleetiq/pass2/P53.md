### P53-F01: Complete absence of test coverage for critical_battery.jsp and CriticalBatteryServlet
**Severity:** CRITICAL
**Description:** The critical_battery.jsp page and its backend CriticalBatteryServlet have zero test coverage. This page handles critical battery level monitoring for vehicles, processes user authentication via session, extracts customer/location/department parameters, and makes 9+ distinct AJAX calls to various servlet endpoints. No tests verify parameter extraction, session handling, permission checks, or servlet responses.
**Fix:** Create comprehensive test suite including: (1) Unit tests for CriticalBatteryServlet covering all 'part' parameter values (cust, permision, dates, table, update); (2) Integration tests verifying correct data flow between JSP and servlet; (3) Session validation tests; (4) Permission-based access control tests; (5) Edge case tests for null/empty parameters; (6) XSS protection tests for user inputs in start_time/end_time fields.
**File:** pages/admin/statistics/critical_battery.jsp
**Line(s):** 1-473
**Untested function:** All server-side parameter extraction (lines 4-6), all AJAX endpoint handlers in CriticalBatteryServlet, all JavaScript functions (getChart, resetPage, buttonFn)

### P53-F02: Null pointer and array bounds vulnerability without test coverage in critical_battery.jsp
**Severity:** CRITICAL
**Description:** Lines 4-6 perform `substring(2)` operations on session attributes without null checking. If `session.getAttribute()` returns null or a string with length < 2, a NullPointerException or StringIndexOutOfBoundsException will be thrown. Lines 222-224 access `permision[manage_vehicle][0/1/2]` without verifying the array dimensions. No tests exist to verify these error paths.
**Fix:** Add defensive null checks before substring operations. Add array bounds validation before accessing permission array. Create test cases specifically for: (1) Null session attributes; (2) Session attributes with length < 2; (3) Missing permission array; (4) Permission array with insufficient dimensions; (5) Malformed permission values.
**File:** pages/admin/statistics/critical_battery.jsp
**Line(s):** 4-6, 222-224
**Untested function:** Session attribute extraction with substring(2)
