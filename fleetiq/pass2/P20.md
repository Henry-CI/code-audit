### P20-F02: Untested AJAX functions in existing_driver_lst.jsp
**Severity:** HIGH
**Description:** The AJAX functions fetch_site (lines 15-29) and fetch_dept (lines 64-86) have no test coverage. These functions make GET requests to get_site.jsp and get_dept.jsp with customer/site codes passed directly in URL parameters without visible encoding. The stateChanged() and stateChanged1() callback functions (lines 31-62, 88-116) parse XML responses and manipulate DOM without error handling for malformed XML. If the XML response is invalid or rec is null (checked on line 40 and 97), the functions fail silently. No tests verify: (1) proper URL encoding of parameters, (2) handling of AJAX failures, (3) handling of malformed XML responses, (4) XSS prevention when inserting nodeValue into DOM, (5) race conditions when multiple AJAX calls overlap.
